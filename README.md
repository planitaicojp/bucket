# bucket

[Scoop](https://scoop.sh/) bucket for PlanitAI tools.

## Usage

```powershell
scoop bucket add planitaicojp https://github.com/planitaicojp/bucket
scoop install lm
```

## Available Apps

| App | Description |
|-----|-------------|
| [lm](https://github.com/planitaicojp/lm-cli) | CLI tool for LINE Messaging API |
